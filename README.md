# Cisco Network Automation Tool

Python-based CLI automation tool built with Netmiko.


![Automation Demo](<img width="711" height="319" alt="Screenshot 2026-02-12 023908" src="https://github.com/user-attachments/assets/c330a80f-28c1-4b58-811c-2bf74374647d" />
)

## Features
- Multi-device SSH via CSV inventory
- VLAN creation with dynamic port assignment
- User provisioning
- Bulk show command execution
- Automated running-config backups

## Lab Environment
Tested in EVE-NG with Cisco IOS images.

## Requirements
- Python 3.x
- Netmiko
